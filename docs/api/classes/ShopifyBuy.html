ShopifyBuy

Module: shopify
Parent Module: shopify-buy

ShopifyBuy only defines one function buildClient which can be used to build a ShopClient to query your store using the provided accessToken, appId, and domain.

Static Methods

ShopifyBuy.buildClient
(
  • configAttrs
)
ShopClient
public

Create a ShopClient. This is the main entry point to the SDK.

const client = ShopifyBuy.buildClient({
  accessToken: 'bf081e860bc9dc1ce0654fdfbc20892d',
  appId: 6,
  myShopifyDomain: 'your-shop-subdomain.myshopify.com', //Deprecated. Use domain instead
  domain: 'embeds.myshopify.com'
});

Parameters:

  • configAttrs Object

    An object of required config data such as: accessToken, appId, domain

    • accessToken String

      An access token for your store. Documentation how to get a token: https://help.shopify.com/api/sdks/js-buy-sdk/getting-started#api-key

    • appId String

      Typically will be 6 which is the Buy Button App Id. For more info on App Id see: https://help.shopify.com/api/sdks/js-buy-sdk/getting-started#app-id

    • domain String

      Your shop's full myshopify.com domain. For example: embeds.myshopify.com

    • myShopifyDomain String

      You shop's myshopify.com domain. [deprecated Use configAttrs.domain]

Returns:

ShopClient:

a client for the shop using your api credentials which you can use to query your store.
