CartLineItemModel

Module: shopify-buy

A cart stores an Array of CartLineItemModel's in it's lineItems property.

Constructor

CartLineItemModel ()

Properties

compareAtPrice

String

Compare at price for variant. The compareAtPrice would be the price of the product previously before the product went on sale. For more info go here.

If no compareAtPrice is set then this value will be null. An example value: "5.00".

grams

Number

Variant's weight in grams. If no weight is set then 0 is returned.

id

String

A line item ID.

line_price

String

The total price for this line item. For instance if the variant costs 1.50 and you have a quantity of 2 then line_price will be 3.00.

price

String

Price of the variant. For example: "5.00".

String
